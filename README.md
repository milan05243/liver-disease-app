# Liver Disease Prediction Web App

## Overview

This is a Streamlit-based web application for predicting liver disease using a machine learning model.

The app allows users to input clinical parameters and get real-time predictions along with insights from the trained model.

---

## Live Demo

https://liver-disease-app-mus9bxy5rbklsj2lykkg4x.streamlit.app/

---

## Features

* User-friendly interface for inputting patient data
* Real-time liver disease prediction
* Integrated machine learning model
* Lightweight and accessible via browser

---

## Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas, NumPy

---

## How to Use

1. Open the live demo link
2. Enter the required clinical values
3. Click on "Predict"
4. View the prediction result

---

## Note

This application is for educational and research purposes only and should not be used for medical diagnosis.
